# AxionAntiAFK
Anti-AFK script that detects player movement, camera movement, and key presses to verify client activity and auto-kick inactive clients.

## Optional Dependency
By defualt, the script is set to use AxionNotifications to send messages to the player. This can be changed in the config to use chat messages. However, if this is option is left at default and you wish to use AxionNotifications, download here: https://github.com/AxionDevelopment/AxionNotifications

## Configuration
Easily customize almost every aspect of the script in the config file located at `/AxionAntiAFK/configs/axionantiafk.config.lua`
